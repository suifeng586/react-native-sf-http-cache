# react-native-sf-http-cache


# 获取react-native自身的缓存大小以及清理




# 安装
* npm install react-native-sf-http-cache




# Methods
|  Methods  |  Params  |  Param Types  |   description  |  Example  |
|:-----|:-----|:-----|:-----|:-----|
|getSize|callBack|null|获取缓存大小(单位byte)||
|clearCache|callBack|null|清除缓存||
